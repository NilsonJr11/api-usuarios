# 🌐 API de Usuários com Flask

API RESTful simples para cadastro e listagem de usuários.

## 🚀 Funcionalidades
- Listar usuários via GET
- Adicionar novos usuários via POST

## 🧰 Tecnologias
- Python 3.x
- Flask

## 📦 Como usar
1. Instale as dependências:
```bash
pip install -r requirements.txt
